Choose your platform

Windows

Windows builds include a portable EXE (contained in a zip archive) that requires no installation, and a MSI installer for more regular use.

Windows builds are only for 64 bit systems.

Linux (Debian/Chromebook)

Linux builds are available for Debian so far, thus being installable on Chromebook. These builds include a portable version, (run with ./bedrockR,) and a DEB installer.

Linux builds are only for ARM64, and AMD64 systems.

ARM64

AMD64

Universal

Universal builds can be run on any device that can run Java 25. Universal builds are (uber-)JAR files that can be run with a installation of Java 25 (bedrockR does NOT support other versions of java.)
